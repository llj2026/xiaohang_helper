# 02 办事流程

> 维护人：小航项目组
> 更新日期：2026-07-15
> 数据来源：郑州航院官网 www.zua.edu.cn / 教务处 jwc.zua.edu.cn / 学生工作部 xsc.zua.edu.cn / 信息整合手册
> 说明：本文件主要服务在校生和教师，覆盖常用办事流程。

---

## 在校证明 / 在读证明

- 【去哪】教务系统自助打印 -> 学院 / 教务处盖章
- 【带什么】学生证（部分场景）
- 【几步】登录 -> 申请 -> 打印 -> 盖章
- 【多久】10 分钟内
- ⚠ 不同用途（签证、银行、单位）模板不同，提前确认对方需要中英文 / 红章

## 成绩单打印

- 【去哪】教务系统（中/英文）-> 教务处档案室盖章（对外用）
- 【带什么】学生证、身份证
- 【多久】当场办结

## 学生证补办

- 【去哪】教务处（本科）/ 研究生院（研究生）
- 【带什么】身份证、近期一寸照、补办申请表（部分要辅导员签字）
- 【多久】1-2 周

## 校园卡补办

- 【去哪】校园卡服务中心
- 【带什么】身份证、工本费
- 【多久】当场新卡 + 余额转入
- 【补充】先用一卡通 App 线上临时挂失防止盗刷 -> 到校园卡服务中心补办

## 转专业申请

- 【时间】大一下学期 / 大二上学期（以教务通知为准）⚠
- 【条件】绩点门槛 + 无违纪 + 面试 / 笔试合格
- 【步骤】
  1. 关注教务通知
  2. 网上申请
  3. 原学院审核
  4. 目标学院考核
  5. 公示
- ⚠ 详见当年《本科生转专业管理办法》

## 学生请销假

- 【3 天内】辅导员审批
- 【3 天以上 / 跨市】学院领导审批
- 【急事 / 病假】先口头报备，事后补办
- 【销假】返校后 24 小时内报辅导员销假
- 【提醒】先请假再走，补假很麻烦

## 户籍迁移（在校期间）

### 迁入
- 凭通知书 + 户口迁移证 -> 报到时交学校 -> 学校汇总到派出所
- 【带什么】身份证、学生证、相关派遣材料

### 迁出
- 毕业前办，有"户口迁回原籍 / 工作地落户 / 留校"三种走向

## 兵役登记 / 应征入伍

- 【兵役登记】每年春季，18 岁以上男生必须线上登记
- 【应征入伍】每年两次征兵，大二大三均可
- 【优待】保留学籍 2 年、退役后学费补偿、退役复学加分等
- 【去哪】学校武装部 / 学生处 / 当地武装部

## 助学金 / 助学贷款

### 家庭困难认定
- 每学年 9 月填《家庭经济困难学生认定申请表》+ 民主评议
- 不再要求乡镇盖章（政策已变，以当年要求为准）⚠

### 国家助学金
- 认定后自动评定，分档发放

### 生源地助学贷款
- 户籍地县区资助中心办理

### 校园地助学贷款
- 学校学生资助中心申请

## 奖学金评定

- 【时间】每年 9-10 月评上学年的奖学金
- 【依据】综合测评（成绩 + 综合素质分）
- 【提醒】保留好获奖证书，综测加分要用

## 勤工助学

- 【去哪】学工部门 / 学院公告栏 / 学生资助中心
- 【说明】优先认定困难学生

## 图书馆借阅

- 校园卡作为借书证
- 本科生通常一次可借 8-10 本，借期 30 天，可续借
- 逾期会停借

## 校园网使用

- 【连接】连接 SSID: 校园 WiFi -> 浏览器自动跳认证页 -> 输入校园网账号（学号）+ 密码
- 【忘记密码】到网信中心 / 信息化办公室凭学生证找回
- ⚠ 详以网信中心通知为准

## 毕业相关流程

- 【流程】毕设开题 -> 中期检查 -> 答辩 -> 论文查重 -> 学位审定 -> 毕业派遣
- 【时间】大四下学期密集进行
- 【证件】毕业证、学位证、报到证、档案、户口

## 挂科 / 重修

- 期末后通常有补考机会
- 补考未过需重修
- 影响奖学金 / 保研 / 学位授予
- 重修需另缴学分费

## 党员发展流程

- 申请书 -> 积极分子 -> 发展对象 -> 预备党员 -> 正式党员
- 全程 2-3 年，大一就可以递交申请书

## 宿舍调整申请

- 向宿管 + 辅导员双线申请
- 说明合理原因，等学院/后勤审批
- 非紧急情况一般不批

## 体测

- 每年秋季学期体测
- 当年有补测机会
- 体测成绩毕业要求（国家标准）
- 长期不及格影响评优

## 教师常用事项

### 新入职教师报到
- 人事处办入职 -> 工资卡 -> 一卡通 -> 教学工号 -> 邮箱 -> 院系报到 -> 党组织关系接转
- 体检、入职培训、廉政教育

### 差旅 / 科研报销
1. 收集合规发票（抬头、税号、内容）
2. 财务一站式服务平台预约 / 线上申报
3. 院系负责人审核 -> 财务复核 -> 打款

### 调课申请
- 网上提交 -> 教学秘书 -> 教务处审批
- 教学事故界定：迟到、调课不报、漏交成绩，后果严重

### 教室设备报修
- 信息化办公室 / 教学楼物业 / 教务处教学保障科报修 ⚠ 以官方为准

### 科研项目立项
- 科技处 / 社科处 / 研究生院，按项目类别走
- 年度申报通知在校园办公网

### 学生考勤异常处理
- 及时录入教务系统 -> 与辅导员沟通 -> 必要时启动学籍预警
- 别等期末

### 监考安排
- 教务处统一排，绝对不能擅自换人

### 教学督导听课
- 不预约，平常心准备
